# Diabetes Prediction

## Problem Statement
Predict whether a patient has diabetes based on 
health metrics like glucose level, BMI, age and 
blood pressure using Machine Learning.

## Dataset
- Source: Kaggle - Pima Indians Diabetes Dataset
- Size: 768 rows, 9 features
- Target: Outcome (0 = No Diabetes, 1 = Diabetes)

## Algorithm Used
K-Nearest Neighbors (KNN)
